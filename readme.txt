this is a billing system project it has three pages.
first page will get the customer email id product id input and quantity of product input  then it has add new button that will add another field to add more products. the in below the user will enter the cash as separate notes and its quantity to add the change in our hand. then amount field that will get the given amount.
in page two the invoice view will appear and denomination calculation will done and the invoice mail will send to the user.
in lookup or history page, can enter the email and get its purchase history.
